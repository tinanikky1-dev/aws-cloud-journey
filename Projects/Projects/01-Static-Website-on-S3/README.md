# Project 1: Host a Static Website on Amazon S3

## Project Overview

This project demonstrates how to host a static website using Amazon S3.

The website introduces me as an aspiring AWS Cloud Engineer and was deployed using Amazon S3 Static Website Hosting.

## Project Objectives

- Create an Amazon S3 bucket
- Upload website files
- Enable Static Website Hosting
- Configure bucket permissions
- Apply a bucket policy
- Make the website publicly accessible

## AWS Services Used

- Amazon S3
- S3 Static Website Hosting

## Project Files

- index.html
- style.css

## Skills Demonstrated

- Creating an S3 bucket
- Uploading website files
- Configuring bucket permissions
- Enabling static website hosting
- Writing an S3 bucket policy
- Hosting a website on AWS

## Outcome

The website was successfully deployed and is publicly accessible through the Amazon S3 Static Website Endpoint.
