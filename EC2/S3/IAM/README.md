# AWS Identity and Access Management (IAM)

## What is IAM?

AWS Identity and Access Management (IAM) is a service that helps you securely control who can access AWS resources and what actions they can perform.

With IAM, you can create users, groups, roles, and policies to manage permissions without sharing your AWS account's root credentials.

## Key Features

- Create and manage AWS users
- Organize users into groups
- Assign permissions using policies
- Create IAM roles for AWS services
- Enable Multi-Factor Authentication (MFA)
- Follow the principle of least privilege

## Common Use Cases

- Managing user access to AWS resources
- Granting permissions to applications
- Providing temporary access using IAM roles
- Securing AWS accounts with MFA

## Benefits of IAM

- Improves security
- Controls access to AWS resources
- Prevents unauthorized access
- Supports secure authentication
