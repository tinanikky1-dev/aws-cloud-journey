# Amazon Virtual Private Cloud (VPC)

## What is Amazon VPC?

Amazon Virtual Private Cloud (VPC) is a networking service that allows you to create your own private network within AWS.

A VPC gives you full control over your network configuration, including IP addresses, subnets, route tables, and security settings.

## Key Features

- Create isolated virtual networks
- Configure public and private subnets
- Control inbound and outbound traffic
- Connect securely to on-premises networks
- Use Internet Gateways and NAT Gateways

## Common Use Cases

- Hosting secure applications
- Building cloud infrastructure
- Connecting AWS to on-premises environments
- Isolating workloads

## Benefits

- Improved security
- Flexible network design
- Better traffic control
- Highly scalable

## What I Learned

- A VPC is a private network in AWS.
- Subnets divide a VPC into smaller networks.
- Public subnets can access the internet.
- Private subnets cannot access the internet directly.
- Route tables determine where network traffic goes.
