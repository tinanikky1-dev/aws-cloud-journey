# Amazon EC2 (Elastic Compute Cloud)

## What is Amazon EC2?

Amazon EC2 is a service provided by AWS that allows users to create virtual servers (called instances) in the cloud.

## Key Features

- Launch virtual servers
- Choose different instance types
- Scale resources when needed
- Secure instances using Security Groups
