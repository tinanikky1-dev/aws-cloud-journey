# Amazon S3 (Simple Storage Service)

## What is Amazon S3?

Amazon S3 is a cloud storage service provided by AWS. It allows you to securely store and retrieve files such as documents, images, videos, backups, and application data from anywhere in the world.

## What I Learned

- S3 stands for Simple Storage Service.
- Data is stored in containers called **Buckets**.
- Files stored in S3 are called **Objects**.
- S3 is highly durable, scalable, and secure.
- It can be used to store backups, static websites, images, videos, and application files.
- You pay only for the storage and resources you use.
